Documentation

This documentation helps you to customize this html template according to your needs.

Folder & Files

index.html - this file which contains html structure and contents of your website.

assets - this folder consists of all the necessary files(eg. css,images,js,fonts) needed for your website.

  • css :- contains all the stylesheets for the website. Files are minified to improve page load speed. Both minified and un-minified files are available.
  • font :- contains all the fonts used for the website. Fonts are taken from Google fonts
  • img :- contains all the images used for the website. Files are compressed to improve page load speed.
  • js :- contains all the javascript files used for the website. Files are minified to improve page load speed. Both minified and un-minified files are available.

Contents

Currently the site is filled with dummy content. But you can edit the contents in the index.html file with the help of a text editor.

The content should be kept inside the html tags as shown below.

            
              <h1> content </h1>
              <p> content </p>
              <button> content </button>
              <a> content </a>

Images

To add/change images, go to the folder img add/replace your new image. Now in index.html file edit the src attribute of img tag.

eg: assume, you added a file named cake.jpg to the img folder. Now the html code to show the image will as follow

            
              <img src="assets/img/cake.jpg">

NB: Always compress images to lowest size without losing quality to ensure faster page load.

Contact Form

If your are using it as a static website, you can integrate the contact form with any static form submition tools (eg. staticforms ) and can recieve the submission directly to your inbox.

Else use the server side code to implement form submission

Pro Tip

Create a copy of index.html file to avoid any kind of data loss while editing the html file.
